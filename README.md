# HeySara

HeySara is a simple Text-to-Speech (TTS) web application that converts written text into spoken audio. It provides a clean and user-friendly interface where users can type any text into an input field and click a button to hear it spoken aloud.

## ✨ Features

* 📝 Simple text input field
* 🔊 Convert text to speech with a single button click
* 🎨 Clean and minimal user interface
* ⚡ Fast and easy to use

## 🚀 How It Works

1. Enter the text you want to hear in the input field.
2. Click the **Speak** button.
3. The application converts the text into speech and plays the audio.

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Web Speech API (or your preferred Text-to-Speech API)

## 💡 Usage

* Click the **Speak** button.
